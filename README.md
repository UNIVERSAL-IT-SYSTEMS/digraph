# digraph

Digraph module provides several functions for working with
[directed graph](http://en.wikipedia.org/wiki/Directed_graph)s (digraphs)
in JavaScript.

## node.js
```js
var digraph = require('digraph');
var g = digraph();
```

## browser

Use browserify.

## API

### g.add.node(attr)

### g.add.edge(attr)

### g.get.nodes()

### g.get.edges()

### g.get.in.edges(node)

### g.get.out.edges(node)
